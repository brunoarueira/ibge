Unofficial gem with a bunch of data collected by IBGE. The data collected are responsibility of IBGE (Instituto Brasileiro de Geografia e Estat√≠stica, in english Brazilian Institute of Geography and Statistics) and this gem makes an integration with ruby more easy.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ibge'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ibge
